# Egg-Timer
Countdown timer using interrupts and outputs the time on the LCD display. 

Equipments used:
  
  Software 
    
    Atmel Studio 7
  
  Hardware
   
    AVR Simon Board 
    LCD Display 
    Connection Wires
    USB cable
